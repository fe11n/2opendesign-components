# props

| name  | type       | 说明 |
| :---- | :--------- | :--- |
| type  | ButtonType | 类型 |
| size  | SizeT      | 尺寸 |
| shape | ShapeT     | 形状 |

```
enum ButtonType {
  PRIMARY = 'primary',
  OUTLINE = 'outline',
  TEXT = 'text',
  LINK = 'link',
}

enum SizeT {
  LARGE = 'large',
  NORMAL = 'normal',
  SMALL = 'small'
}

enum ShapeT {
  ROUND = 'round',
  NORMAL = 'normal'
}
```
