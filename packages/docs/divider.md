# Divider 分割线

## props

| name            | type                    | 默认值       | 说明                 |
| :-------------- | :---------------------- | :----------- | -------------------- |
| type            | DividerTypeT            | 'solid'      | 可选，分割线类型     |
| direction       | DividerDirectionT       | 'horizontal' | 可选，分割线方向     |
| contentPosition | DividerContentPositionT | 'center'     | 可选，自定义内容位置 |
