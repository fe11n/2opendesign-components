# Divider 分割线

## props

| name            | type                            | 默认值       | 说明                 |
| :-------------- | :------------------------------ | :----------- | -------------------- |
| type            | 'solid' \| 'dashed' \| 'dotted' | 'solid'      | 可选，分割线类型     |
| direction       | 'horizontal' \| 'vertical'      | 'horizontal' | 可选，分割线方向     |
| contentPosition | 'left' \| 'center' \| 'right'   | 'center'     | 可选，自定义内容位置 |
