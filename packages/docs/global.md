# 全局配置

## 方法

| name    | type        | 默认值    | 说明                                 |
| :------ | :---------- | :-------- | :----------------------------------- |
| shape   | ShapeT      | 'normal'  | 形状: large \| normal \| small       |
| size    | SizeT       | 'normal'  | 形状 normal \| round                 |
| type    | ButtonTypeT | 'outline' | 形状 primary \|outline\| text \|link |
| loading | boolean     | false     | 加载状态                             |
